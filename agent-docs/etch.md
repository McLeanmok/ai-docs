# Etch

Etch conventions for Gearyco Agentic Web Design web projects. Docs: https://docs.etchwp.com

## When to use

Load this doc when building, templating, or reviewing pages in Etch during workflow step 5 (Development).

## Stack role

Etch is the visual development environment for WordPress in the Gearyco Agentic Web Design stack. It sits alongside WordPress and Automatic.css — markup and structure are authored in Etch; styling follows [acss.md](acss.md) and [css.md](css.md); design quality follows [design.md](design.md).

The CSS reset is documented in [etch-css-reset.md](etch-css-reset.md). Read it before writing any styles.
